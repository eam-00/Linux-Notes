# XFCE

## Theming
Set theme from the CLI:

``xfconf-query -c xsettings -p /Net/ThemeName -s "Greybird"``

Set icon theme from CLI:

``xfconf-query -c xsettings -p /Net/IconThemeName -s Numix-Circle``
